# Contributing

Thanks for your interest in contributing to SystemManager!

## Development Setup

1. Clone the repo: `git clone https://github.com/harpertoken/course.git`
2. Build: `swift build`
3. Run tests: `swift test` (add tests to Tests/ directory)
4. Lint: `swiftlint` (install via `brew install swiftlint`)

## Code Style

- Follow Swift conventions
- Use SwiftLint for style checking
- Add trailing newlines to files
- Keep lines under 120 characters
- Resolve TODOs before merging

## Testing

- Add unit tests for new features
- Run functional tests with CLI/UI
- Ensure builds pass on macOS 14+

## Pull Requests

- Describe changes clearly
- Reference issues if applicable
- Ensure CI passes (build + lint)
- Squash commits if needed

See [docs/architecture-notes.md](docs/architecture-notes.md) for design details.
