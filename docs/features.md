# Features

- Process Management: Launch, monitor, and control child processes
- Resource Monitoring: CPU, memory, and process-specific metrics
- Policy Enforcement: Configurable limits on CPU, memory, timeout, restarts
- Persistence: Saves process state to JSON
- Daemon Mode: Background monitoring and policy enforcement
- CLI Interface: Command-line tool for management
- SwiftUI Dashboard: GUI for management
- Modular Design: Clean separation with Swift packages
